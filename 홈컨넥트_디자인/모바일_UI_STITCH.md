# 홈케어커넥트 모바일 앱 UI 디자인 — Google Stitch 프롬프트

> 앱 이름: 홈케어커넥트 | 플랫폼: iOS & Android (React Native + Expo)
> 디자인 시스템: "The Curated Sanctuary"
> 대상: 65세 이상 어르신(환자), 30~50대 간호사, 병원관리자, 플랫폼운영자

---

## 디자인 토큰

### 색상
- Primary: #002045 (Deep Navy)
- Secondary: #006A63 (Teal)
- Tertiary: #321B00 (Warm Brown, alerts)
- Error: #B3261E (Red)
- Surface: #F7FAFC (Off-white background)
- Surface Container Low: #F0F4F8
- Surface Container: #E8ECF0
- On-Surface: #1A1C1E (Dark text)
- On-Surface-Variant: #6B7280 (Secondary text)

### 타이포그래피 (System Font)
- Hero Title: 28px, ExtraBold, tracking -0.5
- Page Title: 24px, Bold
- Section Title: 18px, Bold
- Body: 16px, Regular (어르신용은 18px)
- Caption: 12px, Medium
- Overline: 10px, Bold, uppercase, tracking 1.5

### 간격
- Page Padding: 20px horizontal
- Section Gap: 24px
- Card Padding: 20px
- Item Gap: 12px

### 컴포넌트 규칙
- Border Radius: 16px (cards), 12px (buttons), 24px (pills)
- No borders: tonal layering으로 구분
- Shadows: 부드러운 elevation (rgba(0,32,69,0.06))
- Glassmorphism: Tab bar, headers (blur + transparency)
- Gradient buttons: from-primary to-secondary
- Touch target: 최소 48px (어르신용: 56px)

---

## 공통 컴포넌트

### Bottom Tab Bar (Glassmorphism)
- 반투명 배경 (white/80 + backdrop-blur-xl)
- 활성 탭: Secondary 색상 아이콘 + 레이블
- 비활성 탭: On-Surface-Variant/40
- 탭 간 구분선 없음
- Safe area 고려 (하단 패딩)

### Top Header
- 좌측: 페이지 제목 (Bold)
- 우측: 액션 아이콘 (알림 벨 with 배지, 설정 기어)
- 배경: transparent 또는 surface

### Card (Bento Style)
- 배경: white / Surface Container Lowest
- Radius: 16px
- Padding: 20px
- Shadow: 0 4px 24px rgba(0,32,69,0.06)
- No border

### StatCard
- 상단: 아이콘 (teal 배경 원형) + 레이블 (caption)
- 하단: 큰 숫자 (28px Bold) + 변화율 칩

### Badge / Chip
- Pill 형태 (radius 24px)
- 배경: 색상/8 (투명도)
- 텍스트: 색상 Bold, 10-12px

### 8. 간호사 방문수행 (10단계 스텝)

```
┌─────────────────────────┐
│ ← 방문 기록              │
│                         │
│ 김순자 환자님 · 3등급     │
│ 서울 서초구 반포대로 123   │
│                         │
│ [1][2][3][4][5] ← Stepper│
│ 체크인→바이탈→체크→메모→아웃│
│                         │
│ ━━━━━━━━━━━━━━━━━━━━━  │
│ Step 3: 수행 체크리스트    │
│                         │
│ ☑ 혈압 측정              │
│ ☑ 혈당 측정              │
│ ☑ 투약 확인              │
│ ☐ 상처 드레싱            │
│ ☐ 운동 지도              │
│                         │
│                         │
│ [━━━ 다음 단계 ━━━]      │
│                         │
│ [오늘] [환자] [알림] [마이] │
└─────────────────────────┘
```

- Stepper: 수평 5단계, 완료=Secondary, 현재=Primary, 미완=Gray
- 체크리스트: 큰 체크박스 (48px touch target)
- 바이탈 입력: 큰 숫자 입력 필드, 단위 레이블
- 메모: 텍스트 + 카메라 버튼 + 음성메모 버튼
- 다음 단계 버튼: Gradient, 전체 너비, 하단 고정

## 모바일 전용 기능 (웹에 없는 것)

### 간호사 전용
- **GPS 체크인/체크아웃**: 위치 자동 기록, 지도 표시
- **카메라 촬영**: 상처/욕창 사진, 식사 사진
- **음성 메모**: 녹음 → 텍스트 변환 저장
- **오프라인 모드**: 배너 표시 + 로컬 저장 → 동기화
- **동선 최적화**: 지도 위 경로 표시

### 환자 전용
- **식사 사진 촬영**: 카메라 → AI 영양 분석
- **약 사진 식별**: 카메라 → 낱알 식별 API
- **복약 푸시 알림**: 시간 도달 시 자동 알림
- **음성 대화**: STT/TTS 기반 에이전트 대화

### 공통
- **푸시 알림**: Expo Notifications
- **생체 인증**: Face ID / Touch ID (설정)
- **딥링크**: 알림 탭 → 해당 화면으로 이동

## 화면 전환 애니메이션

- 탭 전환: Fade (200ms)
- 스택 Push: Slide from Right (300ms, ease-out)
- 모달: Slide from Bottom (250ms)
- 카드 탭: Scale(0.98) → Scale(1.0)

## 접근성 (어르신 대상)

- 최소 터치 타겟: 48px (어르신용 56px)
- 최소 폰트: 14px (어르신 에이전트: 18px)
- 색상 대비: WCAG AA 이상
- 마이크 버튼: 항상 노출, 64px
- TTS: 에이전트 메시지마다 스피커 아이콘
- 에러 메시지: 쉬운 한국어 ("잠시 문제가 생겼어요")
